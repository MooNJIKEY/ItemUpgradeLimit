# Item Upgrade Mod для Valheim

Этот мод позволяет увеличить максимальный уровень улучшения предметов в Valheim.

## Возможности

- Настраиваемый максимальный уровень улучшения для разных категорий предметов:
  - Оружие (мечи, луки, топоры боевые и т.д.)
  - Броня (шлемы, нагрудники, штаны, плечи)
  - Щиты
  - Инструменты (кирки, топоры для дерева и т.д.)
- Возможность установить индивидуальные уровни для конкретных предметов
- Полностью настраивается через конфигурационный файл

## Установка

1. Убедитесь, что у вас установлен BepInEx
2. Скопируйте `ItemUpgradeMod.dll` в папку `BepInEx/plugins/ItemUpgradeMod/`
3. Запустите игру один раз, чтобы создался конфигурационный файл
4. Отредактируйте файл конфигурации в `BepInEx/config/com.valheim.itemupgrademod.cfg`

## Конфигурация

После первого запуска игры с модом, откройте файл конфигурации:
`BepInEx/config/com.valheim.itemupgrademod.cfg`

### Настройки по категориям:

```ini
[Item Categories]
# Максимальный уровень для оружия
WeaponMaxLevel = 10

# Максимальный уровень для брони
ArmorMaxLevel = 10

# Максимальный уровень для щитов
ShieldMaxLevel = 10

# Максимальный уровень для инструментов
ToolMaxLevel = 10

# Максимальный уровень по умолчанию для других предметов
DefaultMaxLevel = 8
```

### Индивидуальные настройки для предметов:

Вы можете установить конкретный уровень для определенных предметов:

```ini
[Custom Items]
# Формат: ИмяПредмета:Уровень,ИмяПредмета2:Уровень2
# Пример:
ItemLevels = SwordIron:15,HelmetBronze:12,ShieldWood:8
```

## Примеры названий предметов

Вот некоторые названия предметов, которые можно использовать:

**Оружие:**
- SwordIron - Железный меч
- SwordBronze - Бронзовый меч
- AxeBronze - Бронзовый топор
- Bow - Лук
- BowFineWood - Лук из хорошего дерева

**Броня:**
- HelmetBronze - Бронзовый шлем
- HelmetIron - Железный шлем
- ArmorBronzeChest - Бронзовый нагрудник
- ArmorIronChest - Железный нагрудник
- ArmorBronzeLegs - Бронзовые штаны

**Щиты:**
- ShieldWood - Деревянный щит
- ShieldBronzeBuckler - Бронзовый баклер
- ShieldIronTower - Железная башня

**Инструменты:**
- PickaxeIron - Железная кирка
- AxeIron - Железный топор (для дерева)

## Важно!

- Значения уровней должны быть от 1 до 20
- Для улучшения предметов до новых уровней вам понадобятся соответствующие материалы
- Мод работает на клиенте, но для многопользовательской игры рекомендуется установить его на сервере

## Компиляция

Для компиляции мода вам понадобится:
1. .NET Framework 4.8
2. Visual Studio 2019 или новее (или dotnet CLI)

Выполните команду:
```bash
dotnet build ItemUpgradeMod.csproj
```

Скомпилированный файл будет автоматически скопирован в папку плагинов.

## Поддержка

Если у вас возникли проблемы:
1. Проверьте файл логов BepInEx: `BepInEx/LogOutput.log`
2. Убедитесь, что все зависимости установлены корректно
3. Проверьте правильность написания названий предметов в конфигурации
